mycqu.mycqu.access_mycqu¶

mycqu.mycqu.access_mycqu(session: requests.sessions.Session, add_to_header: bool = True) → Dict[str, str]¶

用登陆了统一身份认证的会话在 my.cqu.edu.cn 进行认证

参数

  • session (Session) – 登陆了统一身份认证的会话

  • add_to_header (bool, optional) – 是否将 mycqu 的认证信息写入会话属性，默认为 True

返回

mycqu 认证信息的请求头，当 add_to_header 参数为 True 时无需手动使用该返回值

返回类型

Dict[str, str]
